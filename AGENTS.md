# AGENTS.md

## Project

Shared HOCON test resource repository. Contains:
- Lightbend HOCON test suite configuration files
- Expected JSON output generated by the reference implementation
- Generation tooling (Java/Gradle)

## Guidelines

- testdata/hocon/ files must match the Lightbend HOCON test suite exactly
- expected/ files are generated, not hand-edited — regenerate via `cd generate && ./gradlew run`
- Naming: `<conf-basename>-expected.json` for success, `<conf-basename>-expected-error.json` for expected errors
