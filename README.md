# xx.hocon

Shared test resources for HOCON parser conformance testing across [rs.hocon](https://github.com/o3co/rs.hocon), [ts.hocon](https://github.com/o3co/ts.hocon), and [go.hocon](https://github.com/o3co/go.hocon).

## Structure

- `testdata/hocon/` — Lightbend HOCON test suite `.conf` files (master copy)
- `expected/hocon/` — Expected JSON output generated by the Lightbend reference implementation (typesafe-config)
- `generate/` — Java tool to regenerate expected JSON from conf files

## Usage

Parser repos fetch testdata via:

```bash
make testdata
```

This downloads the latest testdata and expected JSON from this repo.

## Regenerating Expected JSON

Requires Java 17+:

```bash
cd generate
./gradlew run
```
